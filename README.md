# Copilot Super 使用说明

Copilot Super 旨在通过 MCP 协议扩展您的 AI 编程助手能力，为您提供更持久、流畅的无限对话体验。

## 📥 安装与激活

1. **安装插件**  
   打开 VS Code 或 Windsurf 的扩展（Extensions）面板，将下载好的 `.vsix` 文件直接拖入面板即可自动安装。

2. **输入注册码**  
   安装成功后，编辑器顶部会自动弹出输入框。请输入您的注册码并回车，完成产品激活。

---

## 🚀 使用指南

### 1. 首次对话：建立连接
为了激活无限对话模式，第一次提问时请务必按照以下步骤操作：

1. 在 Copilot Super 插件面板中，点击 **“复制前置提示词”** 按钮。
2. 转到 **GitHub Copilot** 的原生聊天窗口（Chat）。
3. **粘贴**刚才复制的内容作为第一条消息，并选择您喜欢的模型（如 GPT-4o, Claude 3.5 Sonnet 等）发送。
   > 💡 **注意**：发送成功后，Copilot 将建立与本插件的通信通道。

### 2. 后续对话：无限模式
一旦连接建立，您就可以转回到 **Copilot Super 插件面板** 进行日常使用了：

- **直接在插件中提问**：后续的所有问题请直接在 Copilot Super 插件的输入框中发送。
- **自动上下文增强**：插件会自动处理提示词前缀，确保 AI 始终处于最佳工作状态。
- **不消耗额度**：在插件面板中进行的交互主要通过本地 MCP 服务中转，有助于节省您的 Copilot 使用配额。

---

## ✅ 常见问题

- **状态检查**：请留意底部状态栏，确保 MCP 服务处于运行状态。
- **重新连接**：如果发现对话响应异常，可以尝试重新执行“首次对话”步骤来重置连接。
